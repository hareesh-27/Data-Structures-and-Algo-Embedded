    There are different types of OS but here we will see specifically about **Embedded operating system**

**Embedded OS**
  - An embedded OS is built to run on a embedded system. Basically this is a system integrated to a larger device to perform specific function.
  - Unlike general computer systems, embedded systems are designed for particular tasks such as monitoring, controlling or processing input from sensors.

    **Key characteristics of embedded systems**
            - It uses very little memory and CPU resources
            - It is application specific ( has only minimal options )
            - Often incl. RTOS features to meet timing requirements.

    **Examples of embedded OSes include:**
                FreeRTOS
                VxWorks
                Embedded Linux
                QNX

**Components of embedded OS**
  - Kernel
  - Task Scheduler
  - Device drivers
  - Memory management
  - Communication stack


    **Kernel**
          - A kernel is the core part of an operating system. It acts as a **bridge between software applications and the hardware of a computer**.
          - The kernel **manages system resources, such as the CPU, memory and devices, ensuring everything works together smoothly and efficiently.**
            It handles tasks like running programs, accessing files and connecting to devices like printers and keyboards.
            An Operating System includes the kernel as its core, but also provides a user interface.

    **Task Scheduler**
          - Manages task execution and prioritisation

    **Device drivers**
          - Interfaces that enable OS to communicate with hardware peripherals

    **Memory management**
          - Manages allocation of limited memory resources

    **Communication stack**
          - Protocols for communication bw devices or networks
